# GiST Index Study

## Описание
GiST (Generalized Search Tree) - инфраструктура для реализации индексов произвольных структур данных. Тестируем на встроенных типах PostgreSQL без внешних расширений.

## Преимущества
- Поддерживает полнотекстовый поиск (TSVECTOR)
- Работает с диапазонами (TSRANGE, INET)
- Поддерживает массивы и сложные структуры
- Расширяемость для пользовательских типов данных

## Ограничения
- Обычно медленнее B-tree для простых равенств
- Больший размер по сравнению со специализированными индексами

## Сценарий использования
Тестирование производительности для:
1. Полнотекстового поиска по документам (ОСНОВНОЙ)
2. Запросов по временным диапазонам
3. Поиска по IP-адресам и сетевым диапазонам
4. Работы с массивами тегов
5. Простых пространственных запросов (через point/box)

## Структура данных
- **documents**: 40,000 записей с текстовым контентом и тегами
- **events**: 25,000 записей с временными диапазонами  
- **network_devices**: 20,000 записей с сетевыми адресами
- **simple_locations**: 15,000 записей с координатами

## Созданные индексы
1. `idx_documents_search_vector_gist` - полнотекстовый поиск
2. `idx_events_period_gist` - временные диапазоны
3. `idx_network_ip_range_gist` - сетевые адреса
4. `idx_documents_tags_gist` - индекс для массивов тегов
5. `idx_locations_coords_gist` - пространственный индекс через point

## Требования
Только встроенные возможности PostgreSQL. Никаких внешних расширений не требуется.

## Ожидаемые результаты
- Значительное ускорение полнотекстового поиска
- Эффективный поиск по диапазонам и пересечениям
- Оптимизация работы со сложными типами данных
- Сравнение GiST vs B-tree для разных сценариев
