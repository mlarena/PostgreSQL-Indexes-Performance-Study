# SP-GiST Index Study

## Описание
SP-GiST (Space-Partitioned Generalized Search Tree) - индекс для нерегулярных структур данных, поддерживающий различные алгоритмы разбиения пространства.

## Преимущества
- Эффективен для нерегулярных структур данных
- Поддерживает различные алгоритмы разбиения:
  - Квадродеревья (геоданные)
  - k-d деревья (многомерные данные)
  - Префиксные деревья (строки)
  - Радиальные деревья (сетевые адреса)
- Хорош для поиска ближайших соседей
- Эффективен для префиксного поиска

## Ограничения
- Менее эффективен для простых равенств по сравнению с B-tree
- Может быть больше по размеру
- Сложнее в настройке и понимании

## Сценарий использования
Тестирование производительности для:
1. Пространственных запросов (точки, прямоугольники)
2. Многомерного поиска (k-ближайших соседей)
3. Префиксного поиска по строкам
4. Сетевых диапазонов (IP-адреса)
5. Поиска пересекающихся объектов

## Структура данных
- **spatial_data**: 30,000 записей с точками на плоскости
- **multidimensional_data**: 25,000 записей с многомерными точками
- **text_data**: 20,000 записей со словами для префиксного поиска
- **network_data**: 15,000 записей с сетевыми адресами
- **bounding_boxes**: 10,000 записей с ограничивающими прямоугольниками

## Созданные индексы
1. `idx_spatial_coordinates_spgist` - квадродерево для точек
2. `idx_multidimensional_point_spgist` - k-d дерево для многомерных данных
3. `idx_text_word_spgist` - префиксное дерево для строк
4. `idx_network_ip_spgist` - радиальное дерево для сетевых адресов
5. `idx_bbox_spgist` - для прямоугольников
6. `idx_spatial_composite_spgist` - составной SP-GiST индекс

## Для сравнения
- `idx_text_word_btree` - B-tree для текста
- `idx_network_ip_btree` - B-tree для сетевых адресов

## Ожидаемые результаты
- SP-GiST эффективнее B-tree для префиксного поиска
- Лучшая производительность для пространственных запросов
- Эффективный поиск ближайших соседей
- Хорошая масштабируемость для нерегулярных данных
